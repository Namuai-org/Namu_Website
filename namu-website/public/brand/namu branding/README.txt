NAMU — Brand Assets
===================

FOLDER STRUCTURE
----------------
svg/logo/     → Vector logos (infinitely scalable, use for web, Figma, Illustrator)
svg/icon/     → Vector icons + app icons
png/logo/     → PNG logos at multiple sizes
png/icon/     → PNG icons at multiple sizes  
jpg/logo/     → JPG logos (for docs, presentations, email signatures)
jpg/icon/     → JPG icons

COLOR VARIANTS
--------------
on-white        → Logo/icon on pure white (#FFFFFF) — Word docs, PDFs, emails
on-cream        → Logo/icon on Harmattan cream (#F7F0E3) — Website, brand board
on-black        → Logo/icon on pure black (#000000) — Merch, dark print
on-ink          → Logo/icon on Namu Ink (#1C1410) — Website dark mode, presentations
on-kola         → Logo/icon on Kola brown (#6B3E1E) — Brand materials, packaging
on-sahel        → Logo/icon on Sahel orange (#E8935A) — Accent moments, banners
transparent-dark  → Dark ink logo, no background — drop onto any light surface
transparent-light → Light logo, no background — drop onto any dark surface

PNG SIZE GUIDE
--------------
_web-1x         → Standard web (360×130 logo / 120×120 icon)
_web-2x         → Retina web — use this as default for websites
_favicon-32     → Browser favicon (icon only)
_web-64         → Small UI use (nav bars, headers)
_web-128        → Medium UI / email signatures
_presentation   → Keynote, PowerPoint, Canva, Google Slides
_print-hq       → PDFs, printed documents, business cards
_512px / 1024px → App store icons

USE CASE QUICK GUIDE
--------------------
Website header      → svg/logo/namu-logo-transparent-dark.svg
Website dark mode   → svg/logo/namu-logo-transparent-light.svg
Favicon             → png/icon/namu-icon-on-cream_favicon-32.png
App icon            → png/icon/namu-icon-app-on-ink_512px.png
Keynote / Canvas    → png/logo/namu-logo-on-cream_presentation.png
PDF / Word doc      → jpg/logo/namu-logo-on-white_print-hq.jpg
Business card       → png/logo/namu-logo-on-cream_business-card.png
               OR   → png/logo/namu-logo-on-ink_business-card.png
Email signature     → jpg/logo/namu-logo-on-white_web-128.jpg
Merch / dark print  → png/logo/namu-logo-on-black_print-hq.png
Figma / Illustrator → any .svg file (open and edit freely)
Adobe Illustrator   → open .svg → File → Save As → .ai

FONTS USED
----------
Wordmark: Playfair Display (Google Fonts — free)
https://fonts.google.com/specimen/Playfair+Display
